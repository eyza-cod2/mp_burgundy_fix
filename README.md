# Introduction

The **mp_burgundy_fix** map fixes several bugs in the original **mp_burgundy** map for Call of Duty 2.

## Download
- Actual version:
	- 2021/05/05 - <b><a href="https://github.com/eyza-cod2/mp_burgundy_fix/raw/main/mp_burgundy_fix_v1.iwd">Burgundy FIX v1</a></b>


## Installation
- Download <b><a href="https://github.com/eyza-cod2/mp_burgundy_fix/raw/main/mp_burgundy_fix_v1.iwd">Burgundy FIX v1</a></b> and move it into main folder.
- Change map on the server via command /map mp_burgundy_fix_v1


- Notes:
	- This is first version of the map. Final map name will be mp_burgundy_fix
